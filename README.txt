Nu bör det fungera med SSL- anslutningen. 


1. Gå igenom *.conf i denna mapp för att ändra document root. 


2. Ersätt httpd.conf och http-ssl.conf i /Applications/MAMP/conf/apache(/extra) med de som finns i denna mapp. 

3. Flytta server.crt och server.csr till /Applications/MAMP/conf

4. Starta om MAMP

5. Anslut till CATSHOP på https://localhost
